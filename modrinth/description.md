# Bingo Parachute

`Bingo Parachute` is a server-side Fabric addon for `Yet Another Bingo`.

It replaces the normal ground spawn opening with a configurable airdrop phase at the start of a Bingo match.

Current feature set:

- `BAT` and `ELYTRA` airdrop modes
- Bingo `COUNTDOWN` anchor positioning at high altitude
- Early-match PVP protection
- Inventory and equipment custody during descent
- Timeout handling and fall-damage immunity where needed
- Cleanup and restoration after landing

The project is currently focused on practical server use and iterative testing.
